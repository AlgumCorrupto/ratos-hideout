1. Download the .pnach file
2. Put it in your patches folder (On Windows usually is ``C:\Users\<your-user>\Documents\PCSX2\patches\``)
3. Open PCSX2
4. Right click Midnight Club 3
5. Open properties panel
6. Enable the patch there.

> Beware that the mod only works on MC3R NTSC (american version)
