  • Digital Speedometer (no DLL injection)

    Speedometer that supports KM/H, M/H, and M/S

    Installation Instructions
    1. Download the .pnach file
    2. Put it in your patches folder (On Windows usually is C:\Users\<your-user>\Documents\PCSX2\patches\)
    3. Open PCSX2
    4. Right click Midnight Club 3
    5. Open properties panel
    6. Enable the patch there.

    Beware that the mod only works on MC3R NTSC (american version)

Midnight Club L.A. Remix

  • 8 Player Patch

    Enables 8 players on multiplayer.

    Installation Instructions
    1. Open MCLAR on PPSSPP,
    2. Open the ESC Menu,
    3. Navigate to cheats->edit cheat file,
    4. Copy & Paste the contents of ULUS10383.ini (can be downloaded from the URL)
    5. Reload the game,
    6. Now you can select up to 8 players on multiplayer!
